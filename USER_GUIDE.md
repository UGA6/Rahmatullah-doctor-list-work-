# ADMA Healthcare Chatbot - User Guide

## Welcome Patients! 👋

This guide explains how to use the ADMA Healthcare AI Chatbot to find doctors, get medicine information, and book appointments.

---

## How to Access

### Option 1: Web Browser (Easiest)
Open this address in your browser:
```
http://localhost:8000/ui
```

You will see a friendly chat interface where you can type messages.

### Option 2: API (For Developers)
If you're building an app, use:
```
http://localhost:8000/api/chat/send
```

---

## Quick Start Guide

### Step 1: Open the Chat
Go to http://localhost:8000/ui

### Step 2: Select Language
Click your preferred language at the top:
- English
- اردو (Urdu)
- پښتو (Pashto)

### Step 3: Start Chatting
Type your health concern and press Send or press Enter.

---

## Example Conversations

### Example 1: Finding a Doctor
```
You: I need a doctor
Bot: What symptoms are you experiencing?

You: I have fever and headache
Bot: How long have you had these symptoms?

You: 2 days
Bot: What is your location or city?

You: Karachi
Bot: [Shows list of doctors near you]
```

### Example 2: Medicine Information
```
You: What medicine for fever?
Bot: Paracetamol is commonly used for fever.
     Dosage: 500mg-1000mg every 4-6 hours
     ⚠️ Please consult a doctor first!
```

### Example 3: Lab Tests
```
You: I need a blood test
Bot: What is your location?

You: Lahore
Bot: [Shows list of laboratories in Lahore]
```

## Using Quick Buttons

On the web interface, you can use quick buttons:
- **Find Doctor** - Search for a doctor
- **Medicine Info** - Get medicine information
- **Lab Tests** - Find laboratories
- **Symptoms** - Describe your symptoms

---

## What Can This Chatbot Do?

### 1. Find Doctors
Describe your symptoms, and the chatbot will find the right specialist near you.

**Try saying:**
- "I need a doctor"
- "Find me a cardiologist"
- "I have chest pain"

### 2. Medicine Information
Get information about medicines including:
- What it's used for
- Correct dosage
- Side effects
- Precautions

**Try saying:**
- "medicine for fever"
- "Paracetamol information"
- "what medicine for headache"

### 3. Laboratory Tests
Find laboratories near you for:
- Blood tests
- X-rays
- MRI scans
- And more...

**Try saying:**
- "I need a blood test"
- "Where can I do MRI?"
- "Lab for sugar test"

### 4. Book Appointments
After finding a doctor, you can book an appointment.

## Important Notes

⚠️ **Medical Disclaimer**
- This chatbot provides general information only
- Always consult a real doctor for medical advice
- In case of emergency, call emergency services immediately!

## Troubleshooting

### Chat not responding?
- Make sure the server is running
- Refresh the page

### Can't find your city?
- Try a nearby major city
- Check spelling

### Medicine not found?
- Try different spelling
- Contact support
